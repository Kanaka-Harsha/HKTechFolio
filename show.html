My Projects

ID Card Attendance System

I developed an automated ID card attendance system using RFID, an ESP32, and a PostgreSQL database. A Node.js server updates attendance in real-time, which is then displayed on a web interface built with HTML, CSS, and JavaScript.

IoT Based Smart Forest System

We prototyped an automatic system that monitors forest temperature to detect fires. When a fire is found, it activates a light path, guiding nearby wildlife to safety shelters. The system proposes using sensor nodes and shelters placed throughout the forest.

IoT Based Smart Mirror Demo

We built an IoT smart mirror for restrooms that displays time, plus temperature and humidity when a user is present. It features smoke/fire alerts, Google Assistant voice control for lights, and sends sensor data to ThingSpeak and Adafruit clouds for analysis.
